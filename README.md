## ⚡Como Executar o Projeto⚡

**Pré-requisitos:**
- Node.js
- Java 17
- Expo CLI (opcional para mobile)

### Passo a passo

**Back-End**
```bash
cd Back-End
./mvnw spring-boot:run
```

**Front-End**
```bash
cd Front-End
npm install
npm start
```

**Mobile**
```bash
cd Mobile
npm install
npx expo start
```
